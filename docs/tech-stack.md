Технологический стек:

Frontend:

1) HTML5
2) CSS3
3) JavaScript

Обоснование:
Использование базовых веб-технологий позволяет реализовать интерфейс без избыточной сложности. Это обеспечивает понимание структуры клиентской части, принципов отображения динамического контента и взаимодействия с сервером через HTTP-запросы.



Backend:

1) Язык программирования: Python
2) Веб-фреймворк: Flask

Обоснование:
Flask является лёгким и гибким веб-фреймворком, позволяющим реализовать маршрутизацию, обработку форм, работу с сессиями и создание API. Он подходит для учебных и небольших проектов, обеспечивая прозрачность архитектуры.



База данных:
SQLite

Обоснование:
SQLite используется как встроенная реляционная база данных. Она не требует отдельного сервера и удобна для учебных проектов и прототипирования.



Инфраструктура
1) Backend и frontend могут быть размещены на одном сервере
2) Возможное развёртывание: Render / Railway
3) База данных размещается вместе с backend

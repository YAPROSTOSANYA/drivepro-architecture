Структура приложения:

1) Главная (/) — публичная
2) О школе (/about) — публичная
3) Курсы (/courses) — публичный, динамический список с фильтрацией и пагинацией
4) Детальная страница курса (/courses/:id) — публичная
5) Аутентификация (/auth/login, /auth/register) — вход и регистрация
6) Личный кабинет (/profile) — приватный (избранное, заявки, смена пароля)
7) Запись на курс (/apply) — приватный
8) Админ-панель (/admin) — приватный (только для role=admin)
9) 404 — страница не найдена

Текстовое проектирование интерфейса:

Страница: Главная (/)
Тип доступа: публичный

Основные блоки:
1) Шапка (логотип, навигация, вход/профиль, выход)
2) Hero-баннер с приветствием и кнопками
3) Блок акций и спецпредложений (3 карточки)
4) Блок популярных курсов (3 курса)
5) Блок преимуществ автошколы
6) Блок статистики (выпускники, опыт, инструкторы)
7) Футер

Динамический контент:
- Отображение имени пользователя при авторизации
- Популярные курсы загружаются с сервера


Страница: О школе (/about)
Тип доступа: публичный

Основные блоки:
1) Заголовок страницы
2) Информация об автошколе
3) Список филиалов (4 адреса)
4) Преимущества обучения
5) Контактная информация

Динамический контент:
Статическая информация


Страница: Курсы (/courses)
Тип доступа: публичный

Основные блоки:
1) Заголовок страницы
2) Блок фильтров (поиск, категория, цена, сортировка)
3) Сетка карточек курсов (6 на страницу)
4) Пагинация
5) Кнопка «Подробнее» на каждой карточке

Динамический контент:
- Карточки курсов загружаются с сервера с учётом фильтров и пагинации
- Кнопка «В избранное» (для авторизованных не-админов)


Страница: Детальная страница курса (/courses/:id)
Тип доступа: публичный

Основные блоки:
1) Кнопка «Назад к курсам»
2) Название курса
3) Описание
4) Цена, длительность, категория
5) Время занятий, место проведения
6) Количество свободных мест
7) Кнопка «Записаться на курс» (для авторизованных не-админов)

Динамический контент:
Информация о курсе загружается по его ID


Раздел: Аутентификация
Тип доступа: публичный

Основные блоки:
1) Форма входа (/auth/login) — email, пароль, чекбокс «Запомнить меня»
2) Форма регистрации (/auth/register) — имя, email, пароль, подтверждение пароля
3) Восстановление пароля — модальное окно (email → отправка нового пароля на почту)

Особенность интерфейса:
Регистрация и вход на отдельных страницах, восстановление пароля — в модальном окне

Динамический контент:
- Валидация полей в реальном времени (индикатор сложности пароля)
- Передача данных формы на сервер


Страница: Личный кабинет (/profile)
Тип доступа: приватный (пользователь)

Основные блоки:
1) Информация о пользователе (имя, email, роль, дата регистрации)
2) Кнопка «Сменить пароль» (открывает модальное окно)
3) Блок «Избранные курсы» (карточки с кнопками «Записаться» и «Удалить из избранного»)
4) Блок «Мои заявки» (список заявок со статусом и кнопкой отмены)

Динамический контент:
- Избранные курсы и заявки загружаются с сервера
- Автообновление заявок (polling каждые 5 секунд)


Страница: Запись на курс (/apply)
Тип доступа: приватный (пользователь, не админ)

Основные блоки:
1) Заголовок «Запись на курс»
2) Выпадающий список курсов
3) Кнопка «Записаться»
4) Сообщение об успехе/ошибке

Динамический контент:
- Список курсов загружается с сервера
- Возможен предзаполненный курс через query-параметр ?course_id=


Страница: Админ-панель (/admin)
Тип доступа: приватный (только role=admin)

Основные блоки:
1) Заголовок «Админ-панель»
2) Вкладки: «Управление курсами», «Заявки», «Пользователи»
3) Вкладка «Курсы»: форма добавления курса + список всех курсов с кнопками «Редактировать» и «Удалить»
4) Вкладка «Заявки»: список всех заявок с выпадающим списком для смены статуса
5) Вкладка «Пользователи»: список всех пользователей

Динамический контент:
- Все данные загружаются с сервера
- Автообновление курсов и заявок (polling каждые 5 секунд)
- Модальное окно редактирования курса


Страница: 404
Тип доступа: публичный

Основные блоки:
1) Код ошибки 404
2) Сообщение «Страница не найдена»
3) Кнопки: «На главную» и «Назад»

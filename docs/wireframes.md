Текстовое проектирование интерфейса:

Страница: Главная (/)
Тип доступа: публичный

Основные блоки:
1) Шапка (логотип, навигация, вход/профиль)
2) Приветственный баннер
3) Блок преимуществ автошколы
4) Кнопка перехода к курсам
5) Футер

Динамический контент:
Отображение имени пользователя при авторизации


Страница: О школе (/about)
Тип доступа: публичный

Основные блоки:
1) Заголовок страницы
2) Информация об автошколе
3) Преимущества обучения
4) Контактная информация

Динамический контент:
Статическая информация, может загружаться с сервера


Страница: Курсы (/courses)
Тип доступа: публичный

Основные блоки:
1) Заголовок страницы
2) Список карточек курсов
3) Кнопка перехода на детальную страницу курса

Динамический контент:
Карточки курсов загружаются с сервера


Страница: Детальная страница курса (/courses/:id)
Тип доступа: публичный

Основные блоки:
1) Название курса
2) Описание
3) Стоимость
4) Длительность
5) Кнопка «Записаться»


Динамический контент:
Информация о курсе загружается по его ID

Раздел: Аутентификация (/auth/*)
Тип доступа: публичный

Основные блоки:
1) Форма входа
2) Форма регистрации
3) Восстановление пароля

Динамический контент:
Передача данных формы на сервер


Страница: Личный кабинет (/profile)
Тип доступа: приватный

Основные блоки:
1) Информация о пользователе
2) Кнопка перехода в раздел «Мои заявки»
3) Кнопка выхода


Страница: Запись на курс (/apply)
Тип доступа: приватный

Основные блоки:
1) Форма выбора курса
2) Подтверждение заявки


Страница: Мои заявки (/applications)
Тип доступа: приватный

Основные блоки:
1) Список заявок
2) Статус каждой заявки


Страница: Админ-панель (/admin)
Тип доступа: приватный

Основные блоки:
1) Управление пользователями
2) Управление курсами
3) Управление заявками

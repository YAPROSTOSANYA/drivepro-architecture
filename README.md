# DrivePro

## Описание проекта

DrivePro — это динамическое веб-приложение для автошколы, предназначенное для онлайн-взаимодействия с учениками. Пользователи могут регистрироваться в системе, проходить аутентификацию, просматривать доступные курсы, добавлять их в избранное, записываться на обучение, управлять заявками и своим профилем. Администратор управляет курсами, заявками и пользователями через админ-панель.

Приложение разделено на публичную и приватную части с разграничением доступа. Архитектура построена по клиент-серверной модели. Серверная часть отвечает за обработку запросов, хранение данных и бизнес-логику, клиентская — за отображение интерфейса и взаимодействие с пользователем.

## Целевая аудитория

1. Люди, планирующие получить водительское удостоверение
2. Действующие ученики автошколы
3. Администраторы автошколы

## Ключевой функционал

1. Регистрация и вход в систему
2. Управление пользовательской сессией (http-only cookies)
3. Просмотр списка курсов с фильтрацией, поиском, сортировкой и пагинацией
4. Просмотр детальной информации о курсе
5. Добавление курсов в избранное
6. Запись на обучение с проверкой свободных мест
7. Просмотр и отмена своих заявок
8. Смена пароля в личном кабинете
9. Восстановление пароля через email
10. Админ-панель (управление курсами, заявками, пользователями)
11. Автообновление данных (polling) на страницах профиля и админ-панели

## Основные API эндпоинты

### Аутентификация
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация нового пользователя |
| POST | `/api/auth/login` | Вход в систему |
| GET | `/api/auth/me` | Получение текущего пользователя |
| POST | `/api/auth/logout` | Выход из системы |
| POST | `/api/auth/forgot-password` | Восстановление пароля (отправка нового на email) |

### Курсы
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/courses` | Список курсов (пагинация, фильтры, сортировка) |
| GET | `/api/courses/all` | Все курсы без пагинации |
| GET | `/api/courses/:id` | Детальная информация о курсе |
| POST | `/api/courses` | Создание курса (админ) |
| PUT | `/api/courses/:id` | Обновление курса (админ) |
| DELETE | `/api/courses/:id` | Удаление курса (админ) |

### Избранное
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/favorites` | Список избранных курсов пользователя |
| POST | `/api/favorites/:id` | Добавить курс в избранное |
| DELETE | `/api/favorites/:id` | Удалить курс из избранного |

### Заявки
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/applications` | Заявки текущего пользователя |
| POST | `/api/applications` | Создать заявку на курс |
| DELETE | `/api/applications/:id` | Отменить заявку |

### Админ-панель
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/admin/applications` | Все заявки (админ) |
| PUT | `/api/admin/applications/:id` | Обновить статус заявки (админ) |
| GET | `/api/admin/users` | Список пользователей (админ) |

## Инструкция по локальному запуску проекта

### Требования
- Python 3.8 или выше
- Git

### Шаг 1: Клонирование репозитория
```bash
git clone https://https://github.com/YAPROSTOSANYA/drivepro-architecture.git
cd drivepro-architecture
```

### Шаг 2: Установка зависимостей
```bash
pip install flask flask-sqlalchemy flask mail
```

### Шаг 3: Запуск приложения
``` bash
python app.py
```

### Шаг 4: Открыть в браузере
```bash
http://127.0.0.1:5000
```

### Шаг 5: Тестовый вход
После первого запуска автоматически создаётся администратор:
- Email: `admin@example.com`
- Пароль: `admin123`

Для обычного пользователя зарегистрируйтесь через форму на сайте.
